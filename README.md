# Assignment — React landing page

Single-page marketing layout built with **React** (Vite), CSS Modules, and a production **Docker** image that serves the static build with nginx.

## Run with Docker (required for submission)

From this folder (the one that contains `docker-compose.yml`):

```bash
docker compose up --build
```

Open [http://localhost:8080](http://localhost:8080).

## Local development (without Docker)

```bash
cd assignment-app
npm install
npm run dev
